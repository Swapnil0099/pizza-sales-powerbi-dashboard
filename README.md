# 🍕 Pizza Sales Analytics Dashboard | SQL + Power BI

## 📌 Project Overview
The **Pizza Sales Analytics Dashboard** is a data analytics and business intelligence project designed to analyze pizza sales data and generate meaningful business insights. The project combines the power of **SQL for data analysis** and **Microsoft Power BI for data visualization** to transform raw transactional data into an interactive dashboard.

The objective of this project is to help businesses understand their sales performance, customer purchasing behavior, and product demand using data-driven insights. By analyzing sales trends, identifying best and worst selling products, and visualizing revenue distribution, the dashboard supports better strategic decision-making.



## 🎯 Project Objectives
- Analyze pizza sales data using structured SQL queries  
- Identify top-performing and low-performing pizza products  
- Calculate important KPIs such as revenue, orders, and average order value  
- Track daily and monthly sales trends  
- Understand customer purchasing behavior  
- Build an interactive Power BI dashboard for business insights  



## 🗄 Data Analysis Using SQL
SQL was used to explore and analyze the dataset before visualization. Various analytical queries were written to extract key metrics and patterns from the sales database.

Key SQL analyses performed include:

- Total revenue calculation  
- Total number of orders  
- Total pizzas sold  
- Average order value  
- Average pizzas per order  
- Top 5 pizzas by revenue  
- Bottom 5 pizzas by revenue  
- Top 5 pizzas by quantity sold  
- Bottom 5 pizzas by quantity sold  
- Top 5 pizzas by total orders  
- Daily order trends  
- Monthly order trends  
- Sales distribution by pizza category  
- Sales distribution by pizza size  

These queries helped transform raw sales data into structured insights that were later used in the Power BI dashboard.



## 📊 Power BI Dashboard
The processed dataset was imported into **Power BI** to build a fully interactive dashboard. The dashboard allows users to quickly explore sales performance through various visualizations and KPI indicators.

### Dashboard Components
- KPI Cards for key metrics  
- Daily trend of total orders  
- Monthly sales trend visualization  
- Top 5 best selling pizzas  
- Bottom 5 worst selling pizzas  
- Sales distribution by pizza category  
- Sales distribution by pizza size  
- Interactive slicers and filters for better analysis  

The dashboard design focuses on clarity, usability, and effective data storytelling.



## 📈 Key Performance Indicators (KPIs)

- **Total Revenue:** 817.86K  
- **Average Order Value:** 38.31  
- **Total Orders:** 21,350  
- **Total Pizzas Sold:** 49,574  
- **Average Pizzas per Order:** 2.32  



## 🔍 Key Business Insights
- The **Classic category** contributes the highest portion of total sales.  
- **Large-sized pizzas** generate the maximum revenue.  
- Orders are highest during **weekends**, especially **Friday and Saturday evenings**.  
- Some pizza varieties consistently perform poorly in terms of revenue and order quantity.  
- Monthly trends show peak order activity during certain seasons, indicating potential marketing opportunities.
