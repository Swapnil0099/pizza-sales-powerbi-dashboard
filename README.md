# 🍕 Pizza Sales Analytics Dashboard | Power BI Project

## 📊 Project Overview
The **Pizza Sales Analytics Dashboard** is an interactive business intelligence project built using **Microsoft Power BI**. The goal of this project is to transform raw pizza sales data into meaningful insights that help understand business performance, customer behavior, and product demand.

The dashboard provides a complete overview of sales performance including revenue, order trends, top-selling pizzas, and category-wise distribution. With interactive filters and visualizations, users can explore the data dynamically and gain deeper insights.


## 🎯 Key Objectives
- Analyze overall **sales performance**
- Identify **best and worst selling pizzas**
- Track **daily and monthly order trends**
- Understand **customer ordering patterns**
- Visualize **sales distribution by category and pizza size**
- Support **data-driven decision making**



## 📈 Key Metrics
- **Total Revenue:** 817.86K  
- **Average Order Value:** 38.31  
- **Average Pizzas Per Order:** 2.32  
- **Total Orders:** 21,350  
- **Total Pizzas Sold:** 49,574  



## 🔍 Dashboard Insights

### ⭐ Best Sellers
- **The Thai Chicken Pizza** generates the highest revenue.
- **The Classic Deluxe Pizza** leads in total orders and quantity.

### ⚠ Worst Sellers
- **The Brie Carre Pizza** contributes the lowest revenue, quantity, and orders.

### 📅 Order Trends
- Orders are highest on **weekends**, especially **Friday and Saturday evenings**.
- **July and January** show the highest monthly order volume.

### 🥇 Sales Performance
- **Classic category** contributes the highest share of total sales.
- **Large size pizzas** generate the most revenue.
